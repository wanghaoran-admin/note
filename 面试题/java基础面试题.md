### 1.什么是面向对象？

面向对象 **【OOP】** ，是一种程序范式，它通过将数据和行为封装在对象中的方式来构建软件系统。面向对象的核心概念包括类、对象、继承、多态和封装等。





### 2.JDK、JRE、JVM区别和联系

**JDK：**JDK 是 JRE 的超集，它包含了 JRE，因此在安装 JDK 后，你也可以运行 Java 程序。JDK 是开发人员使用的完整开发环境，而 JRE 则是给最终用户的运行环境。



**JRE：**JRE 提供了执行 Java 程序所需的所有环境和资源，但它不包含开发工具（如编译器）。



**JVM：**JVM 是运行 Java 字节码的虚拟机，它负责将编译后的字节码解释为机器指令，并在操作系统上执行。



**三者联系：**

**JDK > JRE > JVM**。JDK 包含 JRE，JRE 包含 JVM。
