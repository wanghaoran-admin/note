# 第一章 Arduino简介

Arduino 是一个开源的电子原型平台，由硬件和软件两部分组成。它的设计目的是让电子项目开发变得更加简单和易于上手，尤其适合初学者、创客和专业工程师。



### 1. **Arduino 平台的起源**

**起源**：Arduino 项目起源于意大利，由 Massimo Banzi 和 David Cuartielles 等人于 2005 年发起，最初目的是为设计学生提供一个简易、便宜且灵活的电子开发平台。

**开源精神**：Arduino 采用开源硬件和软件的方式，允许任何人自由地使用、修改和扩展，从而在全球范围内形成了一个庞大的用户社区。



### 2.Arduino 硬件

**开发板**：Arduino 硬件的核心是各种型号的开发板，常见的包括 Arduino Uno、Arduino Nano、Arduino Mega 等。每种开发板都有一个微控制器芯片（如 ATmega328P）和一组输入/输出引脚，用于连接其他电子元件。

# 第三章 基本编程语法





### 1.变量

#### **变量的定义**

在 Arduino 中，变量是通过指定数据类型和变量名来定义的。常见的数据类型包括 `int`、`float`、`char`、`boolean` 等。定义变量的基本语法如下：

```
数据类型 变量名 = 初始值;
```

例如：

```
int ledPin = 13;     // 定义一个整型变量，名称为 ledPin，初始值为 13
float temperature = 23.5;  // 定义一个浮点型变量，名称为 temperature，初始值为 23.5
char myChar = 'A';   // 定义一个字符变量，名称为 myChar，初始值为 'A'
boolean isOn = false; // 定义一个布尔型变量，名称为 isOn，初始值为 false
```

####  **变量的类型**

- **int**：用于存储整数，范围通常为 -32,768 到 32,767，占用 2 个字节。
- **float**：用于存储浮点数（小数），精度为 6 到 7 位，占用 4 个字节。
- **char**：用于存储单个字符，范围为 -128 到 127，占用 1 个字节。
- **boolean**：用于存储布尔值 `true` 或 `false`，占用 1 位。



### 2.程序结构

#### `setup()` 函数

- 只运行一次，用于初始化设置。
- 常用于设置引脚模式、初始化串口通信等。

```
cpp复制代码void setup() {
  pinMode(LED_BUILTIN, OUTPUT); // 设置内置 LED 引脚为输出模式
  Serial.begin(9600); // 初始化串口通信，波特率为 9600
}
```







#### **`loop()` 函数**

- 持续运行，用于执行主程序逻辑。
- 常用于读取传感器数据、控制输出等。

```
cpp复制代码void loop() {
  digitalWrite(LED_BUILTIN, HIGH); // 点亮 LED
  delay(1000); // 延迟 1 秒
  digitalWrite(LED_BUILTIN, LOW); // 熄灭 LED
  delay(1000); // 延迟 1 秒
}
```





#### **全局声明**

在 `setup()` 和 `loop()` 之前，您可以声明全局变量、定义常量、以及包含必要的库。这些全局声明在整个程序中都可以使用。

```
int ledPin = 13;         // 全局变量
const int sensorPin = A0; // 常量
```





#### **库的导入**

Arduino 提供了大量的库来简化与硬件的交互。在程序的最顶部，可以通过 `#include` 指令导入所需的库。

```
include <Wire.h>    // 导入 I2C 通信库
#include <Servo.h>   // 导入控制舵机的库
```





#### **自定义函数**

除了 `setup()` 和 `loop()`，你可以创建自己的函数来组织代码。这样可以提高代码的可读性和可维护性。

```
void blinkLed(int pin, int delayTime) {
  digitalWrite(pin, HIGH);  // 打开 LED
  delay(delayTime);         // 等待
  digitalWrite(pin, LOW);   // 关闭 LED
  delay(delayTime);         // 等待
}

void loop() {
  blinkLed(ledPin, 1000);  // 调用自定义函数
}
```
